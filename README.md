# tvBook_v1

A cool and clean web app for binge watchers, find the information for your favourite shows.

## Available Commands

- npm install
- npm start

## Installation

tvBook_v1 requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and start the server.

```sh
cd tvBook_v1
npm i
npm start
```
